# Domain-Driven Design Handbook

A practical, evolving reference for learning and applying Domain-Driven Design (DDD).

The goal of this repository is to build a concise, practical handbook that explains DDD concepts through clear explanations, diagrams, examples, and implementation patterns.

## Run locally

```bash
pip install -r requirements.txt
mkdocs serve
```

http://127.0.0.1:8000
